## 專案描述

**專案名稱**：Mood Movie｜電影情緒探索網站

**目標**：使用 Nuxt 3 串接 TMDB API，打造以「情緒」作為入口的電影探索網站，
讓使用者能依照當下心情瀏覽不同類型的電影推薦。

**核心技術棧**：
- Nuxt 3（SSR）
- TailwindCSS（樣式）
- GSAP（動畫）
- Pinia + LocalStorage（狀態管理與收藏功能）
- TypeScript（全專案）
- TMDB API（電影資料來源）

**核心功能**：
1. 情緒選擇入口首頁（8 種情緒：開心/悲傷/興奮/放鬆/浪漫/刺激/勵志/懷舊）
2. 依情緒篩選的電影列表頁（對應 TMDB genre ID）
3. 電影詳情彈窗（Modal，含劇情簡介、評分、上映日期）
4. 收藏功能（愛心按鈕，Pinia store + LocalStorage 持久化）
5. 收藏清單頁
6. GSAP 動畫（卡片進場 stagger、情緒切換轉場、Modal 開關動畫）

## 規範摘要（詳細請看 CLAUDE.md）

- 全專案 TypeScript，禁用 `any`
- Composition API + `<script setup>` only
- localStorage 存取必須在 `onMounted` 或 `if (process.client)` 內
- GSAP 初始化必須在 `onMounted` 內
- TMDB API Key 不得放在 `runtimeConfig.public`，需透過 server route 代理
- 所有 TMDB 請求封裝於 `composables/useTMDB.ts`
- API 失敗必須顯示錯誤狀態與重試按鈕
- 電影圖片路徑：`https://image.tmdb.org/t/p/w500` + `poster_path`

---

## 視覺風格方向

- **整體氛圍**：沉浸式、電影感、暗色系為主
- **字體**：標題使用有個性的 Display 字體，內文清晰易讀
- **色彩**：每種情緒有對應主色，切換時背景漸層跟著變化
- **卡片設計**：電影海報為主視覺，hover 時顯示簡介與收藏按鈕
- **動畫**：流暢但不過度，情緒切換要有儀式感
